🤖 About File Organizer Bot

File Organizer Bot is a powerful yet simple tool designed to help you organize your messy folders automatically. It categorizes your files by type and moves them into organized subfolders, making it easy to find what you need.

🚀 Features

🎯 Smart Categorization: Automatically sorts files by type (PDFs, Images, Documents, etc.)
🛡️ Safe Operation: Never deletes files - only moves them to organized folders
🔄 Duplicate Handling: Renames duplicate files automatically to prevent overwriting
🌐 Web Interface: Easy-to-use browser-based interface for convenience
📊 Detailed Reports: Shows exactly what was organized and where

⚙️ Technical Details

🐍 Built with Python: Uses Flask for the web interface and standard libraries for file operations
🌐 Local Server: Runs on http://localhost:5000 for security and privacy
🔒 Privacy First: All processing happens locally on your computer
📱 Responsive Design: Works on desktop and mobile browsers

💡 How to Use

  1. Enter the full path to the folder you want to organize
  2. Click "Run Organizer" and wait for the process to complete
  3. Review the results and find your organized files in the new "Organized_Files" folder
  4. Enjoy your newly organized digital space! 🎉
