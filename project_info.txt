🚀 HOW TO USE:

DESKTOP VERSION:
1. Double-click "run_organizer.bat" (Windows)
   OR
   Run "python file_organizer_bot.py"
2. Select folder to organize in GUI dialog
3. Files are automatically sorted

WEB INTERFACE VERSION:
1. Double-click "run_web_organizer.bat" (Windows)
   OR
   Run "python app.py"
2. Open browser to http://localhost:5000
3. Enter folder path in the web form
4. Click "Run Organizer" and view results
